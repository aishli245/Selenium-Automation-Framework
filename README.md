# Selenium Automation Framework 🚀

## 📌 Overview

This project is a Selenium WebDriver Automation Framework built using Java, TestNG, and Extent Reports following the Page Object Model (POM) design pattern.

---

## 🛠️ Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* Extent Reports

---

## 📂 Project Structure

* base → WebDriver setup
* pages → Page Object classes
* tests → Test cases
* utils → Reports & Screenshot utilities

---

## ▶️ How to Run

1. Clone the repository
2. Import as Maven Project in Eclipse
3. Run TestNG tests

---

## 📊 Features

* Page Object Model (POM)
* Extent Reports with logs
* Screenshot on failure
* Reusable automation framework
